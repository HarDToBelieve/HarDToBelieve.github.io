26// We always assume 96 CSS pixels in a CSS inch. This is the cold hard truth of the Web.
27// At high DPI, we may scale a CSS pixel, but the ratio of the CSS pixel to the so-called
28// "absolute" CSS length units like inch and pt is always fixed and never changes.
29const float cssPixelsPerInch = 96;
