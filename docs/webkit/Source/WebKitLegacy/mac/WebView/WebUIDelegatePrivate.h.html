179/*!
180 @method webView:addMessageToConsole:withSource:
181 @param webView The WebView sending the delegate method.
182 @param message A dictionary representation of the console message.
183 @param source Where the message came from. See WebConsoleMessageXMLMessageSource and other source types.
184 @discussion The dictionary contains the following keys:
185
186 <dl>
187 <dt>message</dt>
188 <dd>The message itself.</dd>
189 <dt>lineNumber</dt>
190 <dd>If this came from a file, this is the line number in the file this message originates from.</dd>
191 <dt>sourceURL</dt>
192 <dd>If this came from a file, this is the URL to the file this message originates from.</dd>
193 <dt>MessageSource</dt>
194 <dd>
195 Where the message came from. XML, JavaScript, CSS, etc.
196 See WebConsoleMessageXMLMessageSource and similar constants.
197 </dd>
198 <dt>MessageLevel</dt>
199 <dd>
200 Severity level of the message. Debug, Log, Warning, etc.
201 See WebConsoleMessageDebugMessageLevel and similar constants.
202 </dd>
203 </dl>
204*/
205- (void)webView:(WebView *)webView addMessageToConsole:(NSDictionary *)message withSource:(NSString *)source;
