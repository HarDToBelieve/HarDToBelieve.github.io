76/*
77 Key:
78 A: JSArray*
79 Aap: ArrayAllocationProfile*
80 Ap: ArrayProfile*
81 Arp: BinaryArithProfile*
82 B: Butterfly*
83 By: ByValInfo*
84 C: JSCell*
85 Cb: CodeBlock*
86 Cli: CallLinkInfo*
87 D: double
88 F: CallFrame*
89 G: JSGlobalObject*
90 I: UniquedStringImpl*
91 Icf: InlineCallFrame*
92 Idc: const Identifier*
93 J: EncodedJSValue
94 Mic: JITMathIC* (can be JITAddIC*, JITMulIC*, etc).
95 Jcp: const JSValue*
96 Jsc: JSScope*
97 Jsf: JSFunction*
98 Jss: JSString*
99 L: JSLexicalEnvironment*
100 O: JSObject*
101 P: pointer (char*)
102 Pc: Instruction* i.e. bytecode PC
103 Q: int64_t
104 R: Register
105 Re: RegExp*
106 Reo: RegExpObject*
107 S: size_t
108 Sprt: SlowPathReturnType
109 Ssi: StructureStubInfo*
110 St: Structure*
111 Symtab: SymbolTable*
112 Sym: Symbol*
113 T: StringImpl*
114 V: void
115 Vm: VM*
116 Ws: WatchpointSet*
117 Z: int32_t
118 Ui: uint32_t
119*/
