64 // This method is called whenever style has been computed for an element and the appearance
65 // property has been set to a value other than "none". The theme should map in all of the appropriate
66 // metrics and defaults given the contents of the style. This includes sophisticated operations like
67 // selection of control size based off the font, the disabling of appearance when certain other properties like
68 // "border" are set, or if the appearance is not supported by the theme.
69 void adjustStyle(RenderStyle&, const Element*, const RenderStyle* userAgentAppearanceStyle);

71 // This method is called to paint the widget as a background of the RenderObject. A widget's foreground, e.g., the
72 // text of a button, is always rendered by the engine itself. The boolean return value indicates
73 // whether the CSS border/background should also be painted.
74 bool paint(const RenderBox&, ControlStates&, const PaintInfo&, const LayoutRect&);
75 bool paintBorderOnly(const RenderBox&, const PaintInfo&, const LayoutRect&);
